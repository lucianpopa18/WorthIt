# WorthIt — Download-ready build

This folder is ready to upload as a static site.

## Files
- `index.html` — main page
- `styles.css` — styles
- `app.js` — calculator logic
- `404.html` — optional branded 404 page
- `favicon.svg` — favicon
- `.nojekyll` — keeps GitHub Pages from applying Jekyll processing

## Deploy
1. Upload **all files in this folder** to the root of your site or repository publishing folder.
2. Make sure `index.html` stays at the top level.
3. Commit and publish.

## GitHub link
The footer currently points to `https://github.com/lucianpopa18`.
Change that in `index.html` if you want it to point to a specific repository.
