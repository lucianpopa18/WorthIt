# WorthIt v2

A premium SaaS-style single-page landing page for the WorthIt purchase decision tool.

## Files
- `index.html` — main landing page
- `styles.css` — full UI styling
- `app.js` — interactive calculator logic
- `404.html` — branded fallback page
- `favicon.svg` — site icon
- `.nojekyll` — avoids GitHub Pages processing issues

## Deploy to GitHub Pages
1. Upload all files to the root of your repository.
2. Make sure the main file is named exactly `index.html`.
3. In GitHub: `Settings -> Pages`
4. Set source to `Deploy from branch`
5. Choose `main` and folder `/ (root)`
6. Save and refresh after a short wait.

## Notes
- The calculator updates live as inputs change.
- Saved analyses are stored in the browser with `localStorage`.
- No backend is required.
